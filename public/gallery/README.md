# Gallery images

Drop your photos here. Paths must match `GALLERY_ITEMS` in `constants.js`.

Suggested folders:

```
public/gallery/
  photos/        → moments & life
  achievements/  → awards, certs, wins
  memories/      → trips, friends, random favorites
```

Example: `public/gallery/photos/01.jpg` → `src: "/gallery/photos/01.jpg"`

Supported: `.jpg`, `.jpeg`, `.png`, `.webp`
